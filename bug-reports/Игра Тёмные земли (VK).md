Название: нерабочие кнопки перехода в группу VK и обращения в поддержку

Описание ошибки: кнопки "Перейти" (в тематическую группу VK) и "Отправить письмо" (в поддержку) не реагируют на клики. Проблема снижает вовлеченность игроков.

Шаги воспроизведения:	
1. Запустить игру и дождаться загрузки
2. Пропустить диалог персонажей, если таковые будут
3. Слева найти значок "Сообщество" и нажать на него
Для iOS:
4. Нажать "Социальные сети" → "Перейти".
5. Нажать "Поддержка" → "Отправить письмо".
Для VK (веб, Yandex Browser):
4. Нажать "Поддержка" → "Отправить письмо" (3+ раза).
   
Ожидаемый результат:	iOS: при нажатие кнопки "Перейти" происходит переход на группу в VKontakte/при нажатии кнопки "Отправить письмо" появляется диалоговое окна с возможностью отправить письмо в поддержку
VKontakte: при нажатии кнопки "Отправить письмо" появляется диалоговое окна с возможностью отправить письмо в поддержку

Фактический результат:	iOS: при нажатии кнопки "Перейти" ничего не происходит/ при нажатии кнопки "Отправить письмо" ничего не происходит
VKontakte: при нажатии кнопки "Отправить письмо" ничего не происходит первые два клика, на третий - предложение открыть ссылку (спам mailto → пустая страница.).

Серьёзность:	Средняя (снижает вовлечённость, особенно для новичков; может привести к путанице или уходу игроков)

Окружение:	VK (веб) Яндекс Браузер Версия 25.12.4.1216 (64-bit), iOS версия 17.5.1 (модель  iPhone 11)
Вложения:	VK: логи из DevTools (показывают цикл спама mailto-ссылок при клике "Отправить письмо". )
